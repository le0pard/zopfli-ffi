# Zopfli-ffi

Ruby wrapper for [Zopfli](https://code.google.com/p/zopfli/) library

## Installation

Add this line to your application's Gemfile:

    gem 'zopfli-ffi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zopfli-ffi
